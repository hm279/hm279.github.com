About ListView’s Maximal Item View

Adapter A: set the size of item views to Integer.MAX_VALUE
Adapter B: set the size of item views to Integer.MAX_VALUE
Listview A: has a HeaderView.
Listview B: no HeaderView, no FooterView.

Listview A setAdapter to Adapter A, the result is non item view to show, it’s empty.
Listview B setAdapter to Adapter B, the result is good, shows item views correctly.

Then,
Adapter A: set the size of item views to Integer.MAX_VALUE - 1, Listview A shows items correctly.

I think the max item views of Listview is up to MAX_VALUE, and the Headerview will be counted as one view. FooterView should be the same as HeaderView.
