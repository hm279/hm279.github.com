Should SQLite fts3 contains non-TEXT data?

After research, It seems shouldn’t.
SQLite virtual table fts3 seems treat non-TEXT data like INTEGER BLOB as TEXT, and build full-text index for them too.(Is this right?)

Any way I think we should move the non-TEXT data and no need searchable TEXT data to an ordinary table. Using INNER JOIN to query when necessary.
