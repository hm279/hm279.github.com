set divider for special listview items

Sometimes, I want to set divider for special listview items, but I don’t want add extra view to display as divider.
Required :

  • your listview item do not have a background

You just need to set listview item’s background to a drawable, which only include a line in its top(or bottom). ClipDrawable can create one for you.
