One bug of GZIPInputStream in android 5.0

When I use GZIPInputStream to decompress gzip file in my app, I found that it work on android 4.3, but not work on android 5.0.2.
It throws exception like that:
Caused by: java.util.zip.DataFormatException: invalid distance too far back
Then I look at the gzip format and the GZIPInputStream source code.
I found out that the mistake was in:
int length = Memory.peekShort(scratch, 0, ByteOrder.LITTLE_ENDIAN) & 0xffff;
It should be:
int length = Memory.peekShort(header, 0, ByteOrder.LITTLE_ENDIAN) & 0xffff;

From the gzip format we can know, if the extra field bit was set, Then it need to read
2 bytes optional extra field length and length bytes optional extra field
The following code segment read 2 bytes to get length.

So
Streams.readFully(in, header, 0, 2);
read 2 bytes, and
int length = Memory.peekShort(scratch, 0, ByteOrder.LITTLE_ENDIAN) & 0xffff;
peek the length.
Haha, scratch—-> header

building a suggestion table for SearchView

The system understands several columns, but only two are required.

_ID(required)
A unique integer row ID for each suggestion.
required for a ListView

SUGGEST_COLUMN_TEXT_1(required)
A string
required for presenting as a suggestion.

SUGGEST_COLUMN_TEXT_2
A string.
that is for two-line format suggestion as a second, smaller line.

SUGGEST_COLUMN_ICON_1
A drawable resource, content, or file URI string.
icon-plus-text format suggestion, with the drawable icon on the left side.

SUGGEST_COLUMN_ICON_2
A drawable resource, content, or file URI string.
icon-plus-text format suggestion, with the drawable icon on the right side.

SUGGEST_COLUMN_INTENT_ACTION
An intent action string.
in searchable configuration, android:searchSuggestIntentAction field is the default value for all suggestions.

SUGGEST_COLUMN_INTENT_DATA
A data URI string.
in searchable configuration, android:searchSuggestIntentData field is the default value for all suggestions.

SUGGEST_COLUMN_INTENT_DATA_ID
A URI path string.
“/” and this value is appended to the data field in the intent.

SUGGEST_COLUMN_INTENT_EXTRA_DATA
Arbitrary data.
This column allows suggestions to provide additional data that is included as an extra in the intent’s EXTRA_DATA_KEY key.

SUGGEST_COLUMN_QUERY
required if suggestion’s action is ACTION_SEARCH, optional otherwise.
This is the data that is used when forming the suggestion’s query, included as an extra in the intent’s QUERY key.

SUGGEST_COLUMN_SHORTCUT_ID
Only used when providing suggestions for Quick Search Box.

SUGGEST_COLUMN_SPINNER_WHILE_REFRESHING
Only used when providing suggestions for Quick Search Box.
